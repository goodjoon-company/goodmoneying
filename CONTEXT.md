# goodmoneying

goodmoneying은 개인 투자자가 시장 데이터, 분석 신호, 전략, 봇을 단계적으로 구축해 투자 의사결정 자동화를 준비하는 제품 맥락이다.

## 언어(Language)

도메인 용어의 단일 기준(source of truth)은 [`UBIQUITOUS_LANGUAGE.md`](UBIQUITOUS_LANGUAGE.md)다.

- 새 제품·아키텍처·계약 설명과 사용자 화면 문구는 용어집의 표준 용어를 사용한다.
- 기존 API 경로, 스키마(schema), 코드 식별자(identifier)는 호환성을 위해 과거 이름을 유지할 수 있으며 용어집의 호환성 별칭에서 추적한다.
- 역사적 검증·인계 기록은 당시 표현을 보존한다.
- 일반적인 투자 후보와 시스템이 계산한 수집 후보군(Collection Candidate Pool)을 구분한다.
