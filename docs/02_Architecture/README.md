# 모듈 설계 문서

이 디렉터리는 모듈별 설계 문서를 둔다. `docs/02_Architecture.md`는 전체 색인과 현재 구조를 설명하고, 세부 책임과 흐름은 이 디렉터리의 모듈 문서로 분리한다.

## 파일 규칙

- 파일명: `<모듈명>.md`
- 문서 제목: `# <모듈명> 설계`
- `docs/02_Architecture.md`의 모듈 색인에서 반드시 링크한다.

## 권장 구조

```md
# <모듈명> 설계

## 책임

## 책임이 아닌 것

## 입력과 출력

## 주요 흐름

## 의존성

## 관련 계약

## 리스크와 후속 작업
```
