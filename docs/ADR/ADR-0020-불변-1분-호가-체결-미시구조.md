# ADR-0020: 불변 1분 호가·체결 미시구조 통계

- 상태: Accepted
- 날짜: 2026-07-17
- 관련 Issue: [#29](https://github.com/goodjoon-company/goodmoneying/issues/29)
- 확장: ADR-0013의 시간·결측, ADR-0019의 저장 전용 불변 물질화

## 맥락

현재 호가 요약은 최신 화면용 투영이고 체결 요약은 요청 시 집계다. 계산 버전, 원천 receipt, 지식 시각과 품질 프런티어가 없어 전략·데이터셋이 같은 값을 재현할 수 없다. 호가 WebSocket은 기본 호가와 원화 시장의 모아보기 `level`을 함께 지원하고, 체결의 `ASK`·`BID`는 공식 문서상 매도·매수 구분일 뿐 공격자나 메이커·테이커 의미를 보장하지 않는다.

## 결정

- 첫 버전은 수집 활성화 이후의 UTC 1분 반개방 구간 `[t,t+1m)`만 지원한다. 다주기와 과거 미수집 데이터 합성은 하지 않는다.
- 호가는 같은 구간의 마지막 `(occurred_at, snapshot_id)` `level=0` 스냅숏을 선택한다. 상위 10개 연속 레벨이 없거나 가격 역전·음수 수량이면 값을 만들지 않는다.
- `spread = best_ask - best_bid`, `mid = (best_ask + best_bid) / 2`, `spread_bps = spread / mid * 10000`으로 계산한다.
- `bid_depth_10`과 `ask_depth_10`은 상위 10단계 수량 합계이며 `orderbook_imbalance_10 = (bid_depth_10 - ask_depth_10) / (bid_depth_10 + ask_depth_10)`이다.
- 체결은 공식 `BID=매수`, `ASK=매도` 표시로만 분류한다. 체결 빈도는 1분당 건수, 매수·매도 불균형은 `(BID 수량 - ASK 수량) / 전체 수량`, 체결 강도는 `BID 수량 / ASK 수량 * 100`이다.
- 분모가 0이면 0·무한대·임의 상한을 만들지 않는다. 계산 상태는 `ready|missing|partial|invalid|undefined`, 원천 품질은 `available|no_trade|missing|unavailable|unverified`로 분리한다.
- 체결 원천 합계는 같은 구간의 원천 캔들 개정 거래량·거래대금과 대사한다. 확정 `no_trade`는 체결 빈도 0을 허용하지만 체결 강도와 불균형은 정의되지 않은 값으로 남긴다.
- 계산은 Decimal 정밀도와 `ROUND_HALF_EVEN`을 고정하고 정의 해시, 부모 결과, receipt·호가·체결·원천 캔들·품질 프런티어, `source_as_of`, `knowledge_at`, 입력·내용 해시를 추가 전용으로 저장한다.
- 수집은 receipt와 정규화 행을 직접 연결하고 미시구조 더티 범위까지 같은 트랜잭션으로 기록한다. 이벤트마다 결과 행을 만들지 않고 상품별 1분 범위를 합쳐 닫힌 구간을 계산하며 늦은 원천은 새 불변 정정을 만든다.
- PostgreSQL REST와 분석 WebSocket은 저장 물질화만 읽는다. 최초 조회의 물질화 ID 상한·상품·단위·범위·`asOf`·계산 버전을 커서에 고정한다. SQLite는 요청 시 계산하지 않는 빈 테스트 호환 응답이다.

공식 의미는 작업 시점의 [Upbit WebSocket 체결 문서](https://docs.upbit.com/kr/reference/websocket-trade)와 [Upbit WebSocket 호가 문서](https://docs.upbit.com/kr/reference/websocket-orderbook)를 기준으로 한다.

## 대안

1. 기존 `market_statistics`에 열을 추가하면 캔들 개정 하나를 현재 입력으로 요구하는 계약과 다중 체결·호가 원천 계보가 충돌한다.
2. 요청 시 계산하면 같은 URL이 저장 시점에 따라 달라지고 읽기 API가 원천 잠금과 쓰기를 유발한다.
3. 모든 이벤트를 불변 물질화하면 고빈도 시장에서 큐와 저장량이 입력 이벤트 수에 비례한다.
4. 체결 강도 분모 0을 큰 수나 0으로 치환하면 전략 임계값과 연구 결과를 왜곡한다.

## 결과

- 미시구조 통계는 캔들 지표와 독립된 불변 버전·계보를 가진다.
- 원천 캔들 대사가 맞지 않는 실시간 체결 누락은 값이 아니라 품질 상태로 드러난다.
- 현재 화면용 원시 `analysis.market`과 재현 연구용 `analysis.microstructure`를 구분한다.
- P2-5 데이터셋은 계산 버전과 원천 프런티어를 고정할 수 있다.
