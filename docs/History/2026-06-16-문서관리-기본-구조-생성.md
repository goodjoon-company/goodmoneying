## 변경 요약

## 영향 문서

- `AGENTS.md`: repo-local 문서관리 규칙
- `docs/README.md`: 문서 지도
- `docs/01_Product.md`: 제품 기준 source of truth
- `docs/02_Architecture.md`: 아키텍처 기준 source of truth
- `docs/02_Architecture/README.md`: 모듈 설계 문서 규칙
- `docs/ADR/`: 결정 기록 규칙과 템플릿
- `docs/Task/`: 당시 실행 단위 규칙. 현재는 Milestone별 요약만 유지한다.
- `docs/Test/`: 검증 증적 규칙과 템플릿
- `docs/History/`: 인계 기록 규칙과 템플릿

## 영향 계약

- `docs/contracts/README.md`
- `docs/contracts/db/README.md`
- `docs/contracts/api/README.md`

## 검증

- `goodjoon-workflow` skill 문서를 읽고 구조 기준을 확인했다.
- `rg --files`로 생성된 문서 목록을 확인했다.
- `git status --short`로 신규 파일 상태를 확인했다.
