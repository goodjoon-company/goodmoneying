# 2026-06-21 백필 계획 제어와 worker 지표 정정

## 변경 요약

- 운영 상태 worker 현황판의 `마지막 수집` 표시를 `마지막 저장 성공`으로 변경했다.
- 백필 worker의 동작중 코인은 현재 실행 중인 단일 백필 계획 기준으로 계산하고, 나머지 pending job/target은 `대기 백필` 보조지표로 분리했다.
- 수집 대상/설정 화면의 승인된 백필 계획 카드에 대상 코인과 백필 시작/종료 일시를 표시했다.
- 승인된 백필 작업을 중지하거나 삭제할 수 있게 API와 화면 버튼을 추가했다.
- `BackfillJob` API 응답에 `targetStartAt`, `targetEndAt`, `targets`를 추가하고, DELETE `/v1/backfill/jobs/{jobId}` 계약을 추가했다.
