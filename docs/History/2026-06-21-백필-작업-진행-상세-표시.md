# 2026-06-21 백필 작업 진행 상세 표시

Date: 2026-06-21
Related Task: `docs/Task/M3.md`
Related PR: TBD

## 변경 요약

- Backfill 관리 화면의 백필 작업 카드에 대상 진행 순번, 완료 대상 수, 현재 대상 코인, 현재 대상 백필 row 수, 결측 구간 처리 수, 예상 요청 수를 표시했다.
- 백필 작업 카드의 `결측 구간` 문구를 `결측 구간 처리`로 바꾸고, 대상 목록 hover 부가정보가 있는 카드에는 정보 아이콘을 표시했다.
- 백필 작업 전체 진행률에 현재 대상의 예상 결측 구간 처리 비율을 반영해 한 대상 처리 중에도 진행률이 증가하도록 했다.
- 운영 상태의 코인별 수집 상태에서 전체 수집 대상 50개를 표시하고, 24H 거래대금 기본 정렬 및 헤더 클릭 정렬을 추가했다.
- 백필 워커(Backfill Worker)가 결측 구간 하나를 처리할 때마다 진행 카운터를 저장하도록 했다.
- `/v1/backfill/jobs` 응답에 작업별 진행 상세 필드를 추가했다.
- `stopped` 상태 백필 작업은 작성일 기준 최근 30일 이내인 것만 목록에 노출한다.
- Backfill 관리 화면의 10초 주기 갱신은 기존 구현을 유지하고 테스트로 확인했다.
