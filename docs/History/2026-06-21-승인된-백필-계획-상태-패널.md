# 2026-06-21 승인된 백필 계획 상태 패널

## 변경 요약

- 수집 대상/설정 화면의 백필 승인 패널에 승인된 백필 계획(Backfill Plan) 목록을 추가했다.
- 승인된 백필 job별로 작업 id, 실행 상태, 데이터 타입, 생성 시각, 진행률을 표시한다.
- 기존 `/v1/backfill/jobs` 응답과 `OperationsSnapshot.backfillJobs`를 사용해 새 API 계약 없이 구현했다.
- 상태 표시는 아이콘(icon)과 한국어 라벨을 함께 제공한다.

## 검증

- `npm --workspace apps/web run test -- src/App.test.tsx`
- `npm test`
- `npm run build`
- `npm run e2e`

## 후속 작업

- 백필 job을 pause/resume/stop 하는 제어 버튼이 필요하면 별도 Task로 분리한다.
