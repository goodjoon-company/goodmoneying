## 변경 요약

- 기존 옵션 기반 `goodmoneying_worker.main` CLI(Command Line Interface)를 제거했다.
- 실시간 수집 워커(Realtime Collection Worker)와 백필 수집 워커(Backfill Collection Worker)를 별도 모듈과 실행 단위로 분리했다.
- 백필 수집 워커는 기본 10초 폴링(Polling)으로 DB의 백필 작업(Backfill Job)을 확인한다.
- 로컬 `dev.sh`, Docker Compose, prod-home Compose와 healthcheck를 두 워커 이름으로 정렬했다.
