# 코인 전용 실시간 분석

Date: 2026-07-14
Related Task: [#12](https://github.com/goodjoon-company/goodmoneying/issues/12)
Related Verification: `docs/Test/2026-07-14-코인-전용-실시간-분석-검증.md`

## 변경 요약

- 기존 화면을 유지하고 새 `코인 분석` 메뉴를 추가했다.
- 관심목록 UI의 주식 전환·주식 예정 메뉴를 제거하고, `MarketListRow.assetType` OpenAPI 계약을 `coin` 단일 값으로 좁혔다.
- 관심목록 코인만 WebSocket 분석 구독을 허용하고 차트·지표·시장 요약을 독립 메시지로 전송한다.
- 1분 원천 데이터에서 일봉을 파생하고 주봉·월봉을 제공한다. 차트에는 거래량, SMA, EMA, 볼린저 밴드, RSI와 호가·체결 요약을 함께 표시한다.
- Nginx와 Vite 프록시의 WebSocket 업그레이드를 보장하고 모바일 메뉴 접근성 회귀를 보완했다.

## 리스크와 후속

- 장기 일봉의 품질은 저장된 원천 캔들 범위에 의존한다. 실제 업비트 수집 운영에서는 백필 완료 상태를 확인해야 한다.

## 2026-07-16 완료 기준 보완

- 관심 코인·시간 단위·기간 변경 시 분석 WebSocket을 다시 연결하지 않고 기존 연결에 새 `analysis.subscribe`를 전송하도록 수정했다.
- API 통합 테스트와 실제 Chromium WebSocket 프레임 검증으로 `1m`, `5m`, `10m`, `30m`, `1h`, `1d`, `1w`, `1M` 전체 시간 단위와 1년·3년 범위를 확인했다.
- BTC에서 ETH로 관심 코인을 변경한 뒤 상품·차트·지표·시장 요약이 독립 메시지로 다시 수신되는지 검증했다.
- 전체 검증 결과는 [코인 전용 실시간 분석 검증](../Test/2026-07-14-코인-전용-실시간-분석-검증.md#2026-07-16-issue-12-완료-기준-보완)에 기록했다.

## 2026-07-16 독립 리뷰 후속 보완

- 구독 전송 시 소켓·선택 키를 기록해 `open` 이벤트와 선택 변경 효과가 겹치더라도 동일 구독을 한 번만 보낸다.
- 이벤트 처리기는 정리 여부와 현재 소켓 일치를 확인해 이전 소켓의 지연 `open`·메시지·오류·종료가 현재 구독을 변경하지 못하게 한다.
- API·Playwright 고정 데이터에 BTC 3개월, ETH 2개월의 구분 가격을 추가해 전체 시간 단위의 실제 캔들·지표, 1년·3년 범위 차이, BTC→ETH 화면 교체를 검증한다.
- 증적은 [독립 리뷰 후속 보완 검증](../Test/2026-07-14-코인-전용-실시간-분석-검증.md#2026-07-16-독립-리뷰-후속-보완)에 기록했다.
