# 업비트 API 테스트 거래쌍과 차트 추가 조회 인계

Date: 2026-07-16
Related Issue: [#14](https://github.com/goodjoon-company/goodmoneying/issues/14)
Related Request: `docs/refinements/requests/2026-07-14-업비트-API-테스트-페이지.md`

## 변경 요약

- 상단 요청 패널에서 거래쌍 목록과 캔들 API를 선택해 요청한다.
- 거래쌍 목록의 상세 정보 옵션과 검색·표·원본 JSON 결과를 제공한다.
- 목록 API 응답으로 기존 거래쌍 텍스트 입력을 선택기로 대체한다.
- 캔들 결과를 차트·보조지표·OHLCV 표·원본 JSON으로 표시한다.
- 차트 범위가 적재 데이터 가장자리에 접근하면 캔들을 추가 요청해 중복 없이 병합한다.
- 브라우저 Origin 요청 제한에 맞춰 전체 업비트 REST 요청을 10초 간격으로 제어한다.

## 경계와 후속 작업

- 내부 API·DB·수집 워커·WebSocket 계약은 변경하지 않았다.
- 업비트 원화 환산 응답은 종가만 별도 표시하며 원래 OHLC를 변조하지 않는다.
- PR 병합 후 기능 브랜치와 작업 트리 정리는 사용자 요청에 따라 별도로 수행한다.
