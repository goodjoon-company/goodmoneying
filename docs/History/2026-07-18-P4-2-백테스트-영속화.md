## 변경 요약

- P4 백테스트 run과 결과 테이블을 DB migration으로 추가했다.
- `input_hash`, `result_hash`, dataset content hash, strategy graph hash를 run에 저장한다.
- published 전략 버전과 sealed 데이터셋 버전만 backtest run 입력으로 허용한다.
- terminal run과 trade/equity/metric/artifact 결과 행을 append-only로 봉인한다.
- `PostgresBacktestStore`가 completed run과 결과 행을 한 트랜잭션으로 저장하고 멱등 replay를 지원한다.
- dbmate migration E2E에 `tests/e2e/test_live_postgres_backtest_runs.py`를 포함했다.
