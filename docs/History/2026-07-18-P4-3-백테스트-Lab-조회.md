# P4-3 백테스트 Lab 조회 인계

날짜: 2026-07-18

## 변경 요약

- OpenAPI에 `GET /v1/backtest-runs/{backtestRunId}`와 Backtest schema를 추가했다.
- FastAPI `create_app()`에 `backtest_repository` 주입점을 추가하고 PostgreSQL 런타임에서는 `PostgresBacktestStore`를 사용한다.
- Backtest Store 조회 응답에 체결 순번, 발생·지식 시각, artifact media type을 포함했다.
- Web `api.ts`에 `BacktestRun` 타입과 `loadBacktestRun()`을 추가했다.
- Operations Console에 Backtest Lab 메뉴를 추가하고 읽기 전용 조회 화면을 구현했다.
- seeded E2E API와 테스트 fixture에 `Run #21` 백테스트 결과를 추가했다.
