## 검증 대상

## 실행 명령

| 명령 | 결과 | 메모 |
|---|---|---|
| `mkdir -p docs` | Pass | 이동 대상 디렉터리를 생성했다. |

| `rg -n '01_Product|02_Architecture|contracts|ADR|Task|Test|History|README\\.md' AGENTS.md docs` | Pass | 주요 경로 표현을 점검했다. |

## 수동 검증

- `AGENTS.md`가 `docs/...` 경로를 source of truth로 가리키는지 확인했다.
- root `README.md`가 상세 문서 지도 대신 `docs/README.md`를 안내하는지 확인했다.
- `docs/Task/`, `docs/Test/`, `docs/History/`에 이번 이동 작업 기록이 남았는지 확인했다.

## 미검증 항목
