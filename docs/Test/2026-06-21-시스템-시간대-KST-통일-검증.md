## 확인 내용

- `minute_bucket("2026-01-01T00:00:30+09:00")`는 `2026-01-01T00:00:00+09:00`을 반환한다.
- 백필 계획 생성 입력 `2026-01-01T00:00`은 API payload에서 `2026-01-01T00:00:00+09:00`으로 변환된다.
- 업비트 외부 API 요청의 `to` 파라미터는 provider protocol에 맞춰 UTC로 보내지만, 응답 row의 `candle_start_at`은 KST ISO 문자열로 변환된다.
- `docker-compose.yml`의 `postgres`, `api`, `realtime-collection-worker`, `backfill-collection-worker`, `web` 서비스에 `TZ=Asia/Seoul`이 설정되어 있다.
- `apps/api/Dockerfile`, `apps/worker/Dockerfile`, `apps/web/Dockerfile`은 `TZ=Asia/Seoul`과 `tzdata` 설치를 포함한다.
- PostgreSQL 서비스는 `postgres -c timezone=Asia/Seoul`로 시작한다.
