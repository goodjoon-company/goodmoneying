## 수동 검증

- 구식 `worker` bash loop 프로세스가 남아 있어 중지했다. 새 백필 수집 워커는 pending 백필 작업을 실제로 처리할 수 있으므로 검증 중 임의로 기동하지 않았다.
- `./dev.sh status` 기준 현재 로컬 런타임은 PostgreSQL, API, web이 실행 중이고, 두 수집 워커는 중지 상태다.

## 미검증 항목

- Docker 이미지 실제 build/push와 prod-home 원격 배포는 실행하지 않았다. 로컬 검증은 compose/profile dry-run 테스트와 healthcheck dry-run 테스트로 대체했다.

## 결론

실시간 수집 워커와 백필 수집 워커 역할 분리, CLI 옵션 제거, 로컬/운영 실행 단위 이름 정렬은 자동화 테스트와 E2E 검증을 통과했다.
