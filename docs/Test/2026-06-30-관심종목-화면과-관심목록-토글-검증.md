# 2026-06-30 관심종목 화면과 관심목록 토글 검증

## 변경 범위

- `시장 리스트` 메뉴를 `관심종목`으로 변경했다.
- `관심종목` 메뉴를 좌측 내비게이션 최상단 1레벨 메뉴로 배치했다.
- 운영 화면 공통 상단에 관심 코인 개수 버튼을 추가하고, 별 아이콘 버튼을 누르면 관심 코인 목록 레이어 팝업(layer popup)을 표시한다.
- 관심종목 화면에 코인/주식 세그먼트 전환을 추가했다.
- 코인 탭은 후보 유니버스(Candidate Universe) 100개를 기준으로 관심 여부, 현재가, 24시간 거래대금, 전일 종가 대비 등락률, 기준일시, 캔들 커버리지, 1분 캔들 수를 표시한다.
- KRW 현재가와 24시간 거래대금은 소수부를 제거해 표시한다.
- 캔들 커버리지 시작일은 저장된 1분봉이 없어도 수집 계획(Collection Plan)의 기준 시작일을 표시한다.
- `/v1/market-list/stream`은 `marketList` SSE(Server-Sent Events) 이벤트로 관심종목 가격 정보를 push한다.
- 관심종목 화면은 `marketList` SSE 이벤트를 받으면 현재가, 24시간 거래대금, 등락률, 기준일시를 갱신한다.
- 관심 별 토글은 기존 활성 수집 대상(Active Collection Target) 저장 API인 `PUT /v1/collection-targets`를 호출한다.
- 관심목록 순서 변경은 기존 활성 수집 대상(Active Collection Target) 저장 API인 `PUT /v1/collection-targets`를 호출하고, 운영 상태, Backfill 관리, 공통 관심 코인 목록이 같은 순서를 사용한다.
- `/v1/market-list` 계약을 관심종목 화면용 View Model로 확장했다.

## 확인 항목

- `시장 리스트` 메뉴는 노출되지 않고 `관심종목` 메뉴가 노출된다.
- `관심종목` 메뉴는 데이터 수집관리 그룹 밖의 최상단 1레벨 메뉴로 노출된다.
- 모든 운영 화면 상단에서 관심 코인 개수 버튼이 노출되고, 클릭하면 관심 코인 목록 레이어 팝업이 열린다.
- 관심종목 화면은 `코인`, `주식` 전환 버튼을 제공한다.
- 관심종목 화면의 별 버튼은 클릭 즉시 관심 추가/관심 제거 상태와 공통 관심 코인 개수를 토글한다.
- 관심종목 화면의 순서 이동 버튼은 관심목록 순서를 바꾸고, 변경된 순서는 공통 관심 코인 목록 레이어에 그대로 반영된다.
- 현재가와 24시간 거래대금은 금액과 화폐 단위를 분리 표시하고, KRW 금액은 소수부를 제거한다.
- 등락률 헤더는 `등락률(전일 종가 대비)`로 표시된다.
- `최신성`은 `기준일시`, `커버리지`는 `캔들 커버리지`, `저장 행`은 `1분 캔들 수`로 대체된다.
- 1분 캔들 수가 0인 행도 캔들 커버리지 좌측에 수집 계획 기준 시작일을 표시한다.
- `marketList` SSE 이벤트를 수신하면 관심종목 화면의 가격 정보가 새 payload 기준으로 갱신된다.
- `품질` 컬럼은 관심종목 화면에서 제거된다.
- 별 토글은 관심목록 변경 요청을 `PUT /v1/collection-targets`로 전송한다.
- 순서 변경은 관심목록 변경 요청을 `PUT /v1/collection-targets`로 전송한다.
