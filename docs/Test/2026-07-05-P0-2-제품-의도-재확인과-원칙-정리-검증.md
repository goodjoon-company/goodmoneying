## E2E(End-to-End) 판단

이번 변경은 런타임 코드와 사용자 화면 동작을 바꾸지 않는 문서 결정이다. 자동화된 브라우저 E2E는 실행 대상이 아니며, P0-4 이후 제품 문서 재작성 또는 화면 동작 변경이 생길 때 실행한다.
