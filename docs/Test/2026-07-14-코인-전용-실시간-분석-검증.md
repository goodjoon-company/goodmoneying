# 코인 전용 실시간 분석 검증

Date: 2026-07-14
Related Issue: [#12](https://github.com/goodjoon-company/goodmoneying/issues/12)

## 검증 범위

- 관심목록 전용 분석 WebSocket과 작은 차트·지표 청크 계약
- 관심목록·OpenAPI가 `coin`만 허용하고 주식 전환·예정 메뉴를 노출하지 않는 제품 경계
- 1분 원천 데이터의 일봉 파생, 주봉·월봉 집계
- 데스크톱과 모바일 분석 메뉴 접근성
- 배포 Nginx의 WebSocket 업그레이드 헤더

## 수동 화면 확인

- 1440px에서 좌측 관심 코인 선택, 시간 단위·기간 제어, 차트·거래량·지표·호가·체결 카드의 정보 위계를 확인했다.
- 390px 모바일에서 상단 가로 메뉴로 `코인 분석`에 진입하고 관심 코인 선택·차트 영역에 접근하는 Playwright 회귀를 추가했다.
- 관심목록 화면에 주식 전환 버튼이나 주식 예정 메뉴가 남지 않는 것을 React 단위 테스트와 Playwright E2E로 확인했다.
- `design-html`, `design-review` skill은 이 호스트에 AskUserQuestion 도구가 없고 작업 트리가 구현 변경으로 더러워 각 skill의 대화형 승인 절차가 차단됐다. 동일한 반응형·접근성·스크린샷 검토 기준은 Playwright 검증으로 적용했다.

## 미검증 항목

- 실제 업비트 외부 API 호출은 기본 자동화 검증에서 의도적으로 제외했다. `GOODMONEYING_LIVE_UPBIT=1` 프로필에서 운영자가 별도로 수집 상태와 저장 범위를 확인한다.
- `goodjoon_workflow_gate.py bundle`은 워크플로 배포 패키지에만 존재하는 `assets/originals`가 소스 트리에 없어 실행할 수 없다. 저장소 코드·문서 검증과는 무관한 패키징 전용 게이트다.

## 2026-07-16 Issue #12 완료 기준 보완

### 보완 범위

- 프론트엔드가 관심 코인·시간 단위·기간 변경마다 WebSocket 연결을 새로 만들지 않고, 기존 연결에 `analysis.subscribe`를 다시 보내도록 연결 수명과 구독 수명을 분리했다.
- API 통합 테스트에서 하나의 연결로 `1m`, `5m`, `10m`, `30m`, `1h`, `1d`, `1w`, `1M` 전체 시간 단위를 구독하고, 각 구독이 `analysis.session`, `analysis.instrument`, `analysis.chart`, `analysis.indicators`, `analysis.market` 독립 메시지를 반환하는지 확인했다.
- Playwright 종단 간 테스트(E2E Test)는 실제 브라우저 WebSocket 프레임을 추적해 기본 1년 일봉, 전체 시간 단위, 월봉 3년 전환, BTC에서 ETH로 관심 코인 변경 후 독립 메시지 수신을 확인한다.

### 코드 리뷰

- 요구사항: Issue #12의 기본 1년 일봉, 최대 3년, 전체 시간 단위, 관심 코인 변경 후 독립 메시지 수신을 단위·API 통합·E2E로 추적한다.
- 아키텍처: 기존 SSE 운영 화면과 업비트 수집 WebSocket은 변경하지 않고 분석 화면의 `/v1/realtime/analysis` 연결만 유지한다.
- 계약·ADR: `analysis.subscribe`의 기존 구독 변경 의미와 ADR-0008의 작은 독립 메시지 결정을 따른다. 메시지 형식과 서버 계약 변경은 없다.
- 보안·성능·운영: 인증·외부 입력 경계를 바꾸지 않는다. 화면 제어 변경마다 연결을 다시 만드는 비용을 제거하고, 연결 종료 시 재연결하면서 최신 구독을 다시 전송한다.
- 디자인·반응형·접근성: 시각 마크업과 CSS는 변경하지 않았다. 기존 데스크톱·모바일 E2E를 포함한 전체 5개 시나리오가 통과했다.
- 독립 리뷰 에이전트는 팀 슬롯 제한(`agent thread limit reached`)으로 시작하지 못했다. 대신 요구사항·아키텍처·계약·ADR·테스트 기준의 직접 리뷰와 전체 자동 검증을 수행했으며 차단 또는 후속 발견사항은 없다.
