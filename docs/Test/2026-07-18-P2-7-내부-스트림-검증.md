## 리뷰 반영

- heartbeat가 미수신 sequence를 앞서 보고하면 gap으로 보고 `snapshot_required`로 전환한다.
- dict가 아닌 JSON 메시지는 handler 예외가 아니라 `analysis.error`로 응답한다.
- 운영 모드에서는 `GOODMONEYING_STREAM_CURSOR_SECRET` 누락을 허용하지 않는다.
- stream subscribe `topic`/`scope`가 payload 문맥과 다르면 `INVALID_TOPIC`으로 거부한다.
