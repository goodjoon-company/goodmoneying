- 없음. WebSocket 지연 네트워크 자체는 TestClient에서 안정적으로 재현하지 않고, 서버 handler의 timeout guard와 `slow_consumer` envelope 경로를 구조 테스트와 계약 테스트로 고정했다.

## 미검증 항목

- P2-8 필수 범위의 미검증 항목은 없다.
- 비차단 보강 권고: REST cursor 발급 뒤 snapshot 내용이 실제 변경되는 통합 테스트와 실제 지연 transport에서 1013 close를 확인하는 비동기 테스트는 P7 운영 품질 검증에서 별도 수행한다.
