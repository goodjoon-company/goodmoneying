## 확인한 동작

- `GET /v1/backtest-runs/{backtestRunId}`는 저장된 run의 성과 지표, 체결, artifact를 반환한다.
- 없는 run은 top-level `BACKTEST_RUN_NOT_FOUND` 404 응답으로 수렴한다.
- Backtest Lab은 `Run #21`, `finalEquity`, `partially_filled`, `walk_forward_summary`를 표시한다.
- Backtest Lab은 실행 생성 버튼을 제공하지 않는 읽기 전용 화면이다.
