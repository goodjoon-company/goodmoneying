# 2026-07-25 P7 WCAG 2.2 AA 검증

## 범위

자동화 가능한 WCAG 2.2 AA proxy로 landmark, 접근 가능한 이름(accessible name), 키보드 초점, 이미지 대체 텍스트, 차트 대체 정보를 검증한다.

## 자동화

- 계약: `docs/contracts/quality/p7-quality-evidence.yaml`의 `accessibility.wcag_2_2_aa`
- 명령: `npm run p7:accessibility`
- 테스트: `tests/e2e/p7-accessibility.spec.ts`
