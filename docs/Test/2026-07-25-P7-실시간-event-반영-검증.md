# 2026-07-25 P7 실시간 event 반영 검증

## 범위

코인 분석 WebSocket의 `analysis.chart` event가 브라우저에 수신된 뒤 차트 UI에 1초 안에 반영되는지 자동 검증한다.

## 자동화

- 계약: `docs/contracts/quality/p7-quality-evidence.yaml`의 `performance.realtime_event`
- 명령: `npm run p7:realtime-event`
- 테스트: `tests/e2e/p7-performance.spec.ts`

## 기준

첫 `analysis.chart` frame 수신 시각부터 `코인 분석 캔들 차트`가 visible이 되는 시각까지 1초 이하여야 한다.

## 결과

| 명령 | 결과 |
|---|---|
| `uv run pytest tests/scripts/test_p7_performance_gates.py -q` | RED, `AssertionError: assert 'planned' == 'passed'` |
| `npm run p7:realtime-event` | PASS, `1 passed (2.4s)`, 테스트 본문 측정 대상은 637ms |
| `npx playwright test tests/e2e/p7-performance.spec.ts` | PASS, `3 passed (4.2s)` |

첫 `analysis.chart` 수신 뒤 브라우저 차트 반영은 1초 목표 이내였다.
