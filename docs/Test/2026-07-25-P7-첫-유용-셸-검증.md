# 2026-07-25 P7 첫 유용 셸 검증

## 범위

브라우저가 `/`에 진입한 뒤 사용자가 다음 행동을 선택할 수 있는 첫 유용 셸(first useful shell)이 3초 안에 표시되는지 자동 검증한다.

## 자동화

- 계약: `docs/contracts/quality/p7-quality-evidence.yaml`의 `performance.first_shell`
- 명령: `npm run p7:first-shell`
- 테스트: `tests/e2e/p7-performance.spec.ts`

## 기준

`코인 분석`, `업비트 API 테스트`, `새로고침` controls가 모두 보이는 시각이 navigation 시작 후 3초 이하여야 한다.
